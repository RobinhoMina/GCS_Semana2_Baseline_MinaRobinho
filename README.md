# GCS Semana 2 – Línea Base v1.0

## Objetivo
Implementar un repositorio como depósito de elementos de configuración y establecer una línea base reproducible y trazable.

## Estructura del repositorio
- docs/: documentación del sistema
- src/: código fuente
- tests/: pruebas
- config/: archivos de configuración
- scripts/: scripts auxiliares

## Creación de la línea base
La línea base se establece mediante un tag Git (v1.0) y un Release asociado.
